# 小寶，對不起 - PWA

Upload all files in this folder to a GitHub repository.

## GitHub Pages
1. Go to your repository settings.
2. Open Pages.
3. Set source to `main` branch and `/root`.
4. Open the GitHub Pages URL on a phone.
5. On iPhone: Share → Add to Home Screen.
6. On Android: Browser menu → Add to Home screen / Install app.

The app works offline after first load and includes built-in sound effects.
